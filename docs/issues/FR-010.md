# [FR-010] Export and Archive Reports

## Labels
`Future`, `enhancement`, `reporting`

## User Story
As a founder, I want to export cycle reports to markdown/JSON so that I can archive progress externally.

## Proposed Solution
### Overview
Add export actions on cycle reports and generate markdown/JSON files in user-selected local paths.

### Implementation Flow
1. Build report serializer for markdown and JSON.
2. Add export action in report detail view.
3. Validate target path and write file with conflict-safe naming.

### Technical Approach
```python
def export_report(report, format, destination):
    payload = to_markdown(report) if format == "md" else to_json(report)
    write_file(destination, payload)
```

### Key Considerations
- Export should not modify stored report state.
- Handle path permission errors gracefully.

## Acceptance Criteria
- [ ] Founder can export any cycle report as markdown or JSON.
- [ ] Exported files include all report sections and metadata.
- [ ] Permission/path errors are surfaced with actionable messages.
