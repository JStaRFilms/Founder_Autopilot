# [FR-006] Dashboard Timeline and Metrics

## Labels
`MUS`, `enhancement`, `frontend`

## User Story
As a founder, I want a dashboard that shows events, focus score trends, and active nudges so that I can inspect what happened and why.

## Proposed Solution
### Overview
Build dashboard pages for daily timeline, score visualization, and notification center backed by local API endpoints.

### Implementation Flow
1. Create timeline component for normalized events.
2. Add score cards and trend chart for daily/weekly view.
3. Build nudge feed with status and dismiss/snooze actions.

### Technical Approach
```tsx
<Dashboard>
  <FocusScoreChart data={focusScores} />
  <ActivityTimeline events={events} />
  <NudgeCenter nudges={nudges} />
</Dashboard>
```

### Key Considerations
- Prioritize fast local load with pagination/virtualization.
- Keep explanations visible (why score changed, why nudge fired).

## Acceptance Criteria
- [ ] Dashboard displays event timeline, focus trend, and nudge center in one flow.
- [ ] Founder can filter timeline by source and date.
- [ ] UI shows reason text for score and nudge changes.
