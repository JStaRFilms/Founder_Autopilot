# [FR-007] Weekly Cycle Reports

## Labels
`MUS`, `enhancement`, `reporting`

## User Story
As a founder, I want cycle reports summarizing wins, drift patterns, and next actions so that I can run lightweight weekly reviews.

## Proposed Solution
### Overview
Generate a weekly `CycleReport` artifact from scored signals and nudge outcomes, then render in dashboard.

### Implementation Flow
1. Implement report generator for weekly window.
2. Aggregate wins, drift patterns, and suggested actions.
3. Expose report history and detail page in dashboard.

### Technical Approach
```python
report = CycleReport(
    averageFocusScore=avg(scores),
    topWins=extract_wins(events),
    driftPatterns=extract_drift(signals),
    recommendedActions=build_actions(signals, nudges),
)
```

### Key Considerations
- Reports must be reproducible for a fixed date window.
- Recommendations should be concise and actionable.

## Acceptance Criteria
- [ ] Weekly report auto-generates at configured cycle boundary.
- [ ] Report includes average focus score, wins, drift patterns, next actions.
- [ ] Founder can view current and prior weekly reports in dashboard.
